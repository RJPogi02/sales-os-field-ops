# Security and privacy

This portfolio edition is a local-first frontend prototype.

- Do not commit real CRM exports, private contact lists, API keys, or `.env` files.
- Do not place unrestricted provider keys directly in frontend code.
- Treat public lead-discovery results as unverified candidates.
- Demo Mode is a visual presentation aid, not encryption or secure redaction.

If you find a security or privacy issue, please report it privately through the repository owner's GitHub profile rather than opening an issue containing sensitive data.
